Play With Aarhus

An interactive map of Aarhus with augmented building blocks to play with the famous buildings of Aarhus.

Prototype

Prototype

The prototype consists of white blocks that can be placed on an interactive table showing a map of Aarhus. When the blocks is placed on the table they are augmented by a projector such that the blocks look like building parts of famous buildings e.g. the Aros rainbow. The blocks can be moved around and placed on top of each other creating new interesting buildings.

System Design

The blocks (A) has a NodeMCU, RFID reader and RFID tag inside to detect how the blocks are stacked and BullsEye-tags on the bottom. The NodeMCU sends signal to the server (B). The computer running the Unity scene (C) receives data from the server and the other pc to create the graphics to project on the blocks. The computer controlling the interactive table (D) is detecting the BullsEye-tags on the blocks and is sending the positions of the blocks to the unity scene pc.

Unity

My main focus in the project was the unity development. The scene was a model of the interactive table and physical blocks and a unity camera located at the same position as the projector according to the interactive table. Based on the data from the server and the interactive table the 3D models of the blocks is placed on the model of the interactive table stacked the same way they are stacked in the real world. In the scene the lighting is getting stronger as more blocks are placed on the table and people are walking around the buildings and inside the Aros rainbow.
