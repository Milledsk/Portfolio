WindO

A shape-changing wind screen to extend the use of urban spaces.

Prototype

Koncept

Many urban spaces by the waterfront have a limited use as they are cold and windy. WindO is a shape-changing wind screen that adapts the wether and creates a pleasing outdoor space.

Prototype

The prototype is a part of the WindO wind screen circle. The four acrylic windows can move independently based on input from a manual controller or a wind sensor.

Technologies

WindO is controlled by an Arduino Mega and a linear actuator for each of the four windows. The position is basen on input from either the wind sensor or the manual controller.

Technologies
