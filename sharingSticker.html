Stick-to-share

An augmented reality app and target manager for ad-hoc file sharing.

Prototype

Koncept

Physical stickers can create access to digital resources such as websites, files, videos etc. The AR app gives easy access to the digital resource assigned each target.

AR app

The AR app is developed with Unity and Vuforia. If a target is assigned a public folder the folder the target is augmented with a folder and all the files within. Each file can be access by clicking the icon.

Target manager

The online target manager makes it possible to upload new targets and assign a digital resource. The user can also update existing targets or delete targets.

Technologies

Unity (C#), Vuforia, HTML, CSS, JavaScript, NodeJS
